# drone-inclinaison
Projet autour des systèmes de drones et de la mesure de l’inclinaison en temps réel.


# Projet drone

## Description du projet

Ce projet explore les systèmes utilisés dans les drones, notamment la mesure de l'inclinaison et la stabilisation.

L'objectif est de comprendre comment les drones utilisent différents capteurs pour se stabiliser en vol.

## Objectifs du projet

- Comprendre la stabilisation d’un drone
- Mesurer l’inclinaison en temps réel
- Étudier les capteurs utilisés dans les systèmes de drones

## Technologies et concepts étudiés

- Capteurs inertiels (IMU)
- Stabilisation
- Systèmes embarqués
- Contrôle en temps réel
